# PM Playground Template

A template repository for a React + Vite playground app. Use this to spin up a new project quickly.

## Getting the template

### Use as a GitHub template

1. Click **Use this template** on the repository page on GitHub.
2. Create a new repository from the template.
3. Clone your new repo:

   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_NEW_REPO.git
   cd YOUR_NEW_REPO
   ```

## Install dependencies

The app lives in the `playground-app` directory. From the repo root:

```bash
cd playground-app
npm install
```

## Run the app

From the `playground-app` directory:

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
